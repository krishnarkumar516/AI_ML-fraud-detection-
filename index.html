AI Credit Card Fraud Detection

Machine Learning based system to detect fraudulent credit card transactions using advanced data analytics.

Project Overview

Credit card fraud detection is an important financial security problem. This project uses machine learning models to analyze transaction data and detect fraudulent activities in real time. The dataset contains **284,807 transactions and 31 features**. The model analyzes transaction patterns and identifies suspicious behavior.

Machine Learning Models Used

Random Forest

Ensemble learning algorithm used for classification. Provides high accuracy and strong fraud detection capability.

Logistic Regression

Statistical model used for binary classification. Efficient for fraud vs non-fraud detection.

Decision Tree

Tree based model that learns decision rules from transaction data. Easy to interpret.

  • Total Transactions : 284,807
  • Fraud Cases : 492

  • Dataset Source : Kaggle

The dataset contains anonymized features obtained using PCA transformation. Only "Time" and "Amount" features are not transformed.
